# Project Ren'Py Translator Tools By NAT - Publish Versions
เครื่องมือแปลภาษาเกมเอ็นจิ้น Ren'Py ซึ่งเป็นเกมแนว Visual Novel ที่สามารถสร้างไฟล์ภาษาผ่าน Ren'Py Editer และปรับแก้โค้ดเพื่อสลับภาษาได้

โปรแกรมนี้ถูกสร้างโดยใช้ AI (GPT,Gemini,Claude) ในการสร้างและออกแบบผ่านเครื่องมือผ่าน Visual Studio Code และ Antigravity พัฒนาเพื่อศึกษาการประยุกต์ใช้ AI และเทคโนโลยีด้านซอฟต์แวร์ โดยมีเป้าหมายเพื่อช่วยลดเวลาและความซับซ้อนในการแปลบทสนทนาจำนวนมาก

โปรแกรมใช้ javascript / HTML / Node.js / Electron (Desktop Application) และส่วนประกอบโมดูลต่างๆ รวมถึงต้องการใช้อินเตอร์เน็ตสำหรับแปลภาษา

ความสามารถของโปรแกรมเบื้องต้น **ต้องการไฟล์ภาษาผ่าน Ren'Py Editer**
- ดึงไฟล์ภาษาในนามสกุล .rpy สามารถเลือกไฟล์อย่างเดียวหรือทั้งโฟลเดอร์
- เลือกได้ว่าภาษาต้นฉบับในไฟล์ จะถูกแปลในภาษาไหนได้บ้าง (เน้นภาษาไทย)
- ระบบ Unmask เพื่อป้องกันไม่ให้แปล ตัวแปร หรือคำที่อยู่ในวงเล็บ {} , []
- ระบบสำรองข้อมูลไฟล์ต้นฉบับก่อนแปล และหลังจากแปลแล้ว
- แสดงสถานะความคืบหน้าแบบเรียวไทม์

วิธีการติดตั้ง
ดาวห์โหลดไฟล์รูปแบบสำเร็จ .exe *รองรับเฉพาะ Windows 10 ขึ้นไป*
1. คลิ๊กติดตั้งโปรแกรม .exe เหมือนโปรแกรมทั่วไป ตำแหน่งที่ติดตั้งจะอยู่ที่ "C:\Users\_User_\AppData\Local\Programs\***"
2. ใช้งานโปรแกรมได้ผ่าน Shortcut บน Desktop หรือในตำแหน่งที่ติดตั้ง
